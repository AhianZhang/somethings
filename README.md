# AI 写作助手

- 使用 deepseek api
- 支持个人博客、微信公众号、小红书、自由风格文章生成
- 纯 Markdown 解析
- 可支持 Markdown 导入、导出

# 使用
1. 需要先到 [deepseek](https://platform.deepseek.com/) 官网申请 key
2. 点击配置，保存 deepseek key
3. 创作
4. 优化
